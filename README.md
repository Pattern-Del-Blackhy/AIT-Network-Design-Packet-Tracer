# AIT Network Design (Cisco Packet Tracer)

## Project Overview
This project involves designing and simulating a simple network for AIT using Cisco Packet Tracer. The institution has two campuses: **Seaview** and **KCC**.

The network was designed to support communication between campuses, assign IP addresses through subnetting, and implement key networking devices.

---

## Objectives
- Design a network for two campuses (Seaview and KCC)
- Subnet a network and assign IP addresses to devices
- Implement network devices such as routers, switches, and servers
- Ensure connectivity across the network

---

## Network Design

### Campuses
- **Seaview Campus**
- **KCC Campus**

Each campus contains:
- PCs
- Switches
- Routers
- Servers 

---

## Technologies Used
- Cisco Packet Tracer
- IP Addressing & Subnetting
- Routing ( Static )
- Switching

---

## Implementation

### 1. Network Addressing
- Network was subnetted to provide IP addresses for all devices
- Each campus assigned a separate subnet

### 2. Devices Used
- Routers
- Switches
- End Devices (PCs)
- Servers

### 3. Configuration
- IP addresses assigned to all devices
- Interfaces configured and activated
- Routing configured for inter-campus communication

---

## Results
- Successful communication between devices in both campuses
- IP addressing correctly implemented
- Network devices configured and functioning
